商城三期由7个独立软件组成，分别是：商城前台网站、商城电子商务管理系统、商城分销升舱 平台、客户关系管理系统(坐席)、百城当日达物流查询网站、百城通APP(物流配送，Android版)、PC手机商城(Android版与iPhone版 )，GPS定位系统。商城三期打通了第三方平台(淘宝等)->商城->ERP(EAS)三个异构体系，信息得以顺畅流转，同时打通了销售->财 务->仓库/物流这三个大的管理体系，管理效率得到提高。
